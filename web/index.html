TryFun BHMAX BLE 控制台

单文件 Web Bluetooth 页面。已集成伸缩 ffb7/0x0c、加热 ffb5/0x05、随机自动模式、通知 ffb8。

伸缩控制

0%

帧格式：[seq, 02, 00, 03, 0c, level, checksum]，写入 ffb7。

加热控制

OFF

来自日志样本 010200030501fa：命令 0x05，level 1/0，写入 ffb5。

自动随机模式

未启动 当前随机强度 0

每个周期在最小/最大强度之间随机取一个整数并发送到 ffb7。可与音频联动并行运行，多个模式同时写入时命令会交错发送。

音频联动

未启动 检测 0% 发送强度 0

默认“律动/节拍 + 只认鼓点/低频”只用约 70-260Hz 的鼓点突增触发，300-3400Hz 人声只会参与抑制，不会参与触发。切到“音量包络”并关闭过滤，可恢复原来一有声音就按响度动的模式。
