单文件 Web Bluetooth 页面。已集成伸缩 ffb7/0x0c、加热 ffb5/0x05、随机自动模式、通知 ffb8。

来自日志样本 010200030501fa：命令 0x05，level 1/0，写入 ffb5。

自动随机模式

未启动 当前随机强度 0

每个周期在最小/最大强度之间随机取一个整数并发送到 ffb7。启动随机模式会自动停止音频联动/模拟音量，避免多个模式抢写。

音频联动

未启动 音量 0% 发送强度 0

先连接 BLE，再点“开始联动”。麦克风模式可选择内置/外置输入；屏幕/系统音频在 Chrome 里通常要选择“Chrome 标签页”并勾选“共享标签页音频”。正倍数：音量越大强度越高；负倍数：超过阈值后音量越大强度越低；低于阈值仍归零。
